
# Track Hood Inventory Full System

## Features
- Online login system
- Admin/Supervisor/Worker roles
- Inventory management
- IMEI tracking
- Barcode & QR scanning support
- Excel export
- Email notifications
- Sales reports

## Deployment
1. Upload project to GitHub
2. Connect GitHub to Render
3. Deploy backend
4. Add PostgreSQL database
5. Deploy frontend
